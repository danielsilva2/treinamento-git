2 - Qual a diferença entre git e github?

Git é um sistema de controle de versão popular. O GitHub é o maior host de código-fonte do mundo e pertence à Microsoft desde 2018.

3 - O que faz o comando commit?

O Git considera cada commitponto de mudança ou "save point". É um ponto no projeto ao qual você pode voltar se encontrar um bug ou quiser fazer uma alteração.

4 - O que faz o comando push?

O comando git push é usado para fazer upload do conteúdo do repositório local para um repositório remoto. Pushing é como você transfere commits do seu repositório local para um repositório remoto.
